# Product Overview

UIT University is a modern educational platform built as a comprehensive university website with integrated content management capabilities. The platform combines a public-facing university website with sophisticated admin tools for content management.

## Core Features

- **University Website**: Complete university presence with courses, faculty, admissions, and contact information
- **CMS Integration**: Sanity-powered content management for dynamic content updates
- **Admin Dashboard**: Drag-and-drop page builder with Supabase backend for custom page creation
- **Interactive Elements**: Student chat functionality, course browsing, faculty profiles, and timeline displays
- **Multi-Platform**: Responsive design supporting desktop and mobile experiences

## Target Users

- **Students & Prospective Students**: Course discovery, faculty information, admissions process
- **Content Administrators**: University staff managing website content through CMS
- **Page Builders**: Admin users creating custom pages with drag-and-drop functionality
- **Faculty**: Profile management and course information updates

## Key Value Propositions

- Modern, Coursera/Udemy-inspired design for educational content
- Real-time content management with live preview capabilities
- Flexible page building system for marketing and informational pages
- Integrated analytics and performance tracking
- SEO-optimized content structure for better discoverability
